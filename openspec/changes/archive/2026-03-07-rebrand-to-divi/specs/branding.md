# Specification: Branding to DIVI

## Overview
Define the system's presentation of the "DIVI" brand and slogan.

## Requirements

### Requirement: Application Identity
The application must present itself as **DIVI** across all user-facing interfaces.

- **Scenario: App Title**
  - **WHEN** the application is installed or running, **THEN** it must show "DIVI" as the app name in the system drawer or taskbar.
  
- **Scenario: UI Headers**
  - **WHEN** the user is on the Home tabs, **THEN** the top header must display "DIVI" as the brand.

### Requirement: Slogan Presentation
The tagline **"Divida sem Dívida"** must be visible to reinforce the product's mission.

- **Scenario: Splash Screen**
  - **WHEN** the app starts, **THEN** the "DIVI" logo must fade in followed by the slogan "Divida sem Dívida" in a subtler font below.

### Requirement: Consistent Aesthetics
The branding must be consistent with the established FinTech Premium design tokens.

- **Scenario: Typography**
  - **WHEN** the "DIVI" brand is displayed, **THEN** it must use the **Inter** font (extra-bold/900 weights for the logo, light/regular for the slogan).
